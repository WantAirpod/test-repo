# 블로그 자동화 프로젝트 설정

## 블로그 포스팅 규칙

### 글 작성 스타일
- 친근한 말투 사용 (예: ~요, ~습니다)
- 이모지를 적절히 활용
- 문단 사이에 빈 줄 추가
- 소제목(##)을 활용해 가독성 높이기

### 필수 포함 내용
- 글 시작에 인사말
- 중간에 개인적인 경험담 추가
- 마무리에 요약 및 추천 문구

### 이미지 관련
- 상품 리뷰시 최소 3장 이상의 이미지 포함
- 이미지 아래에 간단한 설명 추가

### SEO 최적화
- 제목에 핵심 키워드 포함
- 본문에 키워드 자연스럽게 3-5회 반복

## 명령어 단축
- "블로그 올려줘 [주제]" → 네이버 + 티스토리 동시 발행
- "후기 써줘 [URL]" → URL에서 정보 추출 후 후기 작성
